О озере

Азас — озеро в горах Тывы, привлекательное для любителей дикой природы и уединённого отдыха. Маршрут из Кызыла лучше планировать заранее: часть выездов формируется по расписанию или по заявке на личную поездку.

Для групп друзей удобно заказать транзит целиком — в салоне будут только ваши, без посторонних пассажиров.

  • Направление для опытных путешественников и летнего сезона
  • Уточняйте даты выезда в актуальном расписании
  • Снаряжение и запас еды — с собой
